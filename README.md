# Deep Learning Mid-Term Exam 

এই রিপোজিটরিতে আমার ডিপ লার্নিং (Deep Learning) কোর্সের মিড-টার্ম পরীক্ষার সল্যুশন ফাইলটি রয়েছে।

## Description
এই প্রজেক্টে মূলত ডিপ লার্নিংয়ের মৌলিক বিষয়গুলো নিয়ে কাজ করা হয়েছে। এর মধ্যে উল্লেখযোগ্য বিষয়গুলো হলো:
* **Neural Network Architecture:** একটি ৩-লেয়ার বিশিষ্ট নিউরাল নেটওয়ার্ক ডিজাইন করা হয়েছে যেখানে ইনপুট ফিচার, হিডেন লেয়ার এবং আউটপুট নিউরনের গাণিতিক সমীকরণগুলো দেখানো হয়েছে।
* **Forward Propagation:** প্রতিটি লেয়ারের জন্য Forward Equation এবং Sigmoid অ্যাক্টিভেশন ফাংশন ব্যবহার করে চূড়ান্ত প্রেডিকশন বের করা হয়েছে।
* **PyTorch Autograd Experiment:** পাইটর্চ (PyTorch) কীভাবে গ্রেডিয়েন্ট (Gradients) ক্যালকুলেট এবং জমা (Accumulate) করে তা একটি ছোট কোড এক্সপেরিমেন্টের মাধ্যমে দেখানো হয়েছে।

## Key Learnings
* নিউরাল নেটওয়ার্কের গাণিতিক কাঠামো তৈরি।
* PyTorch-এ `backward()` ফাংশন কল করলে কীভাবে গ্রেডিয়েন্টগুলো আগের মানের সাথে যুক্ত হয় এবং কেন `optimizer.zero_grad()` ব্যবহার করা প্রয়োজন।
